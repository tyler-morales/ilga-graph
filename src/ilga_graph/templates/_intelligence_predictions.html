What do P(Advance), P(Law), and Forecast mean?
  • P(Advance) — Chance this bill ever reaches a positive milestone (passes committee, floor, etc.). One model for all bills; not “next step.” So a bill already at “Passed Both” can still show a lower P(Advance) if it looks like bills that often stall.
  • P(Law) — Chance this bill becomes law, given where it is now (stage, momentum, delays). Bills already at Passed Both / Governor usually have high P(Law).
  • Forecast — “Would this bill become law?” using only intro-day info (sponsor, topic, committee). No progress or staleness. Compare to P(Law) to see how much current momentum matters.
